# Reenquadro-Game

🌀 Reenquadro
Reenquadro é um jogo de puzzle experimental desenvolvido como projeto de conclusão de curso. O objetivo é explorar a interseção entre mecânicas de jogo e psicoeducação, utilizando a mudança de perspectiva como uma metáfora para a empatia e o cuidado com a saúde mental.

💡 O Conceito
Inspirado por clássicos como Paper Mario, o jogo desafia o jogador a alternar entre diferentes planos de realidade (2D e 3D). No contexto do Reenquadro, essa mudança não é apenas um recurso técnico, mas uma representação visual de como a forma como percebemos nossos problemas altera nossa capacidade de resolvê-los.

O projeto foca no pilar da Empatia, convidando o jogador a vivenciar situações sob novos ângulos para encontrar saídas que, em uma visão limitada ou "unidimensional", pareciam inexistentes.

🎮 Mecânicas Principais
Troca de Perspectiva: A capacidade de rotacionar ou "desdobrar" o cenário para revelar caminhos ocultos e novas interpretações do ambiente.

Puzzle Narrativo: Os desafios são integrados a uma história que aborda temas de saúde mental com delicadeza e embasamento.

Interatividade Simbólica: Objetos e obstáculos no mundo do jogo servem como metáforas para barreiras cognitivas e emocionais.

🧠 Embasamento Científico
O desenvolvimento foca em garantir que a representação de demandas de saúde mental seja ética, precisa e promova uma experiência de conscientização sem gerar gatilhos negativos.
